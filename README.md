✨ Random Quote Generator
 
A simple and lightweight web app that displays a new random quote each time the user clicks a button.
Built using HTML, CSS, and JavaScript.

🚀 Features

•    📝 Displays random quotes

•    🔄 New Quote button to generate a fresh quote

•    🎤 Author name shown 

•    🎨 Clean UI

•    🌐 Can fetch quotes from an API

•    🔊 Text-to-speech button

•    📤 Share quote to Twitter

•    📋 Copy quote to clipboard

🛠️ Technologies Used

•    HTML5

•    CSS3

•    JavaScript
